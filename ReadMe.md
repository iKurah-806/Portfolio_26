# Webチャットアプリ開発

## 概要

チーム開発で作成したWebチャットアプリです。
授業の一環での開発のためデプロイは行っていないですが、ユーザー同士がコミュニケーションを取ることを想定としたWebアプリケーションです。

## 使用技術

フロントエンド  
  ・HTML  
  ・CSS  
  ・JavaScript  
  
バックエンド  
  ・Java  
  ・Spring Boot  
  
データベース  
  ・PostgreSQL  

## 主な機能

・グループチャット、ダイレクトチャット  
・メッセージの送信  
・ルーム作成、参加、招待  
・ユーザー登録、ログイン  

## 担当部分 

・ルーム作成  
・メッセージの編集、削除  
・画像の送信  
・テーマ変更  
・サイドバーのドラッグ操作  

## 工夫した点

・直感的に操作できるようにシンプルでわかりやすいUI設計を意識しました。  
・ユーザーの好みに合わせてテーマを変更できるようにしました。  

## 苦労した点

・コミュニケーション不足で作業が滞ることがあり、進捗確認やタスク共有を行うことの大切さを学びました。  
・予測していないエラーが発生した際、ログやエラーメッセージを確認して原因を特定して対応しました。  

## 今後の改善

・コードの可読性向上  
・セキュリティの強化  
・チーム開発におけるコミュニケーションの改善
