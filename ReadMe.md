# Webチャットアプリ開発

## 概要

チーム開発で作成したWebチャットアプリです。
授業の一環での開発のためデプロイはしてはいないですが、ユーザー同士がコミュニケーションをとることを想定としたWebアプリケーションです。

## 使用技術

フロントエンド
  ・HTML
  ・CSS
  ・JavaScript
  
バックエンド
  ・Java
  ・Spring Boot
  
データベース
  ・PostgreSQL

## 主な機能

・ユーザー登録、ログイン
・メッセージ、画像送信
・メッセージの編集、削除
・ルーム作成、参加

## 工夫した点

・直感的に操作できるようにシンプルでわかりやすいUI設計を意識しました。
・ユーザーの好みに合わせてテーマを変更できるようにしました。

## 苦労した点

・コミュニケーション不足でさ作業が滞ることがあり、進捗確認やタスク共有を行うことの大切さを学びました。
・予測していないエラーが発生した際、ログやエラーメッセージを確認して原因を特定して対応しました。

## 今後の改善

・コードの可読性向上
・UIの改善
・セキュリティの強化
・チーム開発のにおけるコミュニケーションの改善
